void Toubab.Beinder.Tools.TypeAdapterRegistryTest.FindReturnsInReverseRegistrationOrder1 ( )

Test the order in which types are found and returned

This is important because with this behavior, it is possible to override previous registrations with later ones.

The list should be ordered as follows:

  1. More specialized adapters first (i.e., their adaptee type argument is more specialized)
  2. If two adapters are equally specialyzed, first comes the one that was added last.
