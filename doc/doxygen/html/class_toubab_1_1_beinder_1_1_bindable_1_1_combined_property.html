Detailed Description

Combines multiple IProperty instances into one

This class is used by Scanner.CombinedScanner in case two (or more) scanners return an IProperty instance with the same name for the same object scan.

Multiple IProperty instances are combined as follows.

Broadcasts: All contained properties' broadcast events can result in a broadcast event from this combined property. The combined property only fires a broadcast when a contained broadcast carries a changed set of Values.

Handle Broadcasts: The combined property handles broadcasts that are received by calling TryHandleBroadcast on all contained properties (in order), until one signals that the broadcast has been handled (by returning true).

See CombinedBindable{T} documentation for more information on how IBindable properties and methods on instances are are combined.
