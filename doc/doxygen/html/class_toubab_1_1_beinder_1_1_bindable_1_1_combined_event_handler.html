Detailed Description

Combines multiple IEventHandler instances into one

This class is used by Scanner.CombinedScanner in case two (or more) scanners return an IEventHandler instance with the same name for the same object scan.

Multiple IEventHandler instances are combined as follows.

Handle Broadcasts: The combined property handles broadcasts that are received by calling TryHandleBroadcast on all contained properties (in order), until one signals that the broadcast has been handled (by returning true). If IBindable.ValueTypes of a contained event handler are not compatible with the arguments supplied to TryHandleBroadcast, that method is not called on that instance.

See CombinedBindable{T} documentation for more information on how IBindable properties and methods on instances are are combined.
