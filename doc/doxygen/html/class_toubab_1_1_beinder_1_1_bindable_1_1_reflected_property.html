Detailed Description

Reflected property bindable

Adapts a reflected proeprty (PropertyInfo) to the IProperty interface.

The property's value can be accessed through Values at index 0. In the case of ReflectedProperty, Values will always have a length of 1.

If the property's value changes, an event might be raised to indicate this. This event can be specified in the constructor.

ReflectedProperty can also react to IEvent.Broadcast events through TryHandleBroadcast: it sets the value of the underlying property accordingly. While setting the property, the "change event" handler is temporarily disconnected to prevent Broadcast events emanating from this instance, to prevent Broadcast events caused by other Broadcast events (which would result in an endless loop).
