Detailed Description

Binder: binds properties, events and methods of different objects

The Binder uses a CombinedScanner to harvest IBindable instances (representing properties, events and methods) from one or more objects.

The collection of IBindables is searched for matching property names; bindables with matching property names are bound to each other.

The principal entry point(s) for this class are Bind(object[]) and Bind(IEnumerable{object}).

All private methods are also documented here, because the code is quite complex and of vital importance to Toubab.Beinder.

IBindings Toubab.Beinder.Binder.Bind ( IEnumerable< object >  objects)

Bind properties of given objects.

The Binder uses a Scanner to harvest IBindable instances (representing properties, events and methods) from objects .

The collection of IBindables is searched for matching bindable names; bindables with matching bindable names are bound to each other.

This is done recursively, so that properties of properties are also bound (this can only be the case for bindable properties, represented with IProperty). Furthermore, when a property's value changes, the "properties of properties" (child properties) are automatically rebound.

When the binding is established, values of properties from the first object are propagated to the bound properties on the other objects.
