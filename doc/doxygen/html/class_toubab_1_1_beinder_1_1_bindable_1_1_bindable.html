virtual void SetObject (object value)
 Set the object that this IMixin is attached to. An mixin is "detachted" if the associated object is null, otherwise it is "attached".Whenever a non-null object is set with this method, event handlers might be attached to that object. For that reason, make sure a null object is set whenever possible to avoid obsolete event subscriptions (that e.g. prevent an object from being collected by the garbage collector).
