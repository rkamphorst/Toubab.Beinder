void Toubab.Beinder.Mock.MockProperty.SetObject ( object  newObject)

Set the object that this IMixin is attached to.

An mixin is "detachted" if the associated object is null, otherwise it is "attached".

Whenever a non-null object is set with this method, event handlers might be attached to that object. For that reason, make sure a null object is set whenever possible to avoid obsolete event subscriptions (that e.g. prevent an object from being collected by the garbage collector).

Parameters

bool Toubab.Beinder.Mock.MockProperty.TryHandleBroadcast ( object[]  payload)

Handle a broadcast by IEvent.Broadcast (well, try to).

The payload parameter:

  • Comes from the BroadcastEventArgs.Payload, as broadcast by IEvent.Broadcast.
  • Will be "shared" with other calls, i.e., this array instance is also passed in calls to other IEventHandler instances that handle the same event.
  • Will have at least the length of the array in IBindable.ValueTypes, and each of the types of the payload will be assignable ty the corresponding types in IBindable.ValueTypes.

If BroadcastEventArgs.Payload does not meet these requirements, TryHandleBroadcast will not be called to handle that particular event.

If this method is called, it returns whetner whether the broadcast could be handled by returning true or false.

Returns
true, if the broadcast could be handled by this IEventHandler, false otherwise.
Parameters
payloadPayload from the event.
