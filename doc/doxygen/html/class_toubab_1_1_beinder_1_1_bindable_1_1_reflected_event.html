Detailed Description

Reflected event bindable

Adapts a reflected event (EventInfo) to the IEvent interface.

This class raises Broadcast every time the actual event is raised. The BroadcastEventArgs.Payload will contain the parameters that were passed to the event delegate.

An event handler that raises the Broadcast event is attached to the original event when SetObject is called (with a non-null parameter). If a non-null object was already attached (Object was not null), the event handlers are first detached from that object.

When SetObject is called with a null parameter, event handlers are detached from the Object if it was non-null.

Note that only events with a void return type, and with up to 32 parameters are supported. If you attempt to construct this type with events that don't meet these requirements, an ArgumentException is thrown.
