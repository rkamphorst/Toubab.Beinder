bool TryHandleBroadcast (object[] argument)
 Handle a broadcast by IEvent.Broadcast (well, try to). The payload parameter:
  • Comes from the BroadcastEventArgs.Payload, as broadcast by IEvent.Broadcast.
  • Will be "shared" with other calls, i.e., this array instance is also passed in calls to other IEventHandler instances that handle the same event.
  • Will have at least the length of the array in IBindable.ValueTypes, and each of the types of the payload will be assignable ty the corresponding types in IBindable.ValueTypes.
If BroadcastEventArgs.Payload does not meet these requirements, TryHandleBroadcast will not be called to handle that particular event.If this method is called, it returns whetner whether the broadcast could be handled by returning true or false.

Detailed Description

Reflected event handler bindable

Adapts a reflected event handler (MethodInfo) to the IEventHandler interface.

TryHandleBroadcast will be called in response to a IEvent.Broadcast event. When this happens, the parameters in BroadcastEventArgs.Payload are passed to the method represented by ReflectedBindable{T}.Member.
