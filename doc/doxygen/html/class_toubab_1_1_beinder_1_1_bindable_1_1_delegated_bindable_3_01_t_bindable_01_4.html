 Set the object that this IAnnex is attached to. An Annex is "detachted" if the associated object is null, otherwise it is "attached".Whenever a non-null object is set with this method, event handlers might be attached to that object. For that reason, make sure a null object is set whenever possible to avoid obsolete event subscriptions (that e.g. prevent an object from being collected by the garbage collector).

Detailed Description

Base class for a bindable that delegates to another bindable.

Concrete subclasses of this class are used by Scanner.MixinScanner to delegate to propeties, events and event handlers (methods) on mix-ins.
