Detailed Description

Base class for combined bindables.

Multiple IBindable instances are combined as follows.

Set Object: SetObject results in the object being set for all contained properties.

ValueTypes and Path are all taken from the first event handler in the list.

For more information on how other properties / methods are combined, see the more specific implementations.
