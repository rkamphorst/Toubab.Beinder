Detailed Description

Interface for objects to bind System.Windows.Input.ICommand instances to.

All events on System.Windows.Input.ICommand have method counterparts here, and vice versa. Except for CanExecuteQuery, which has a counterpart Mixin.CommandMixin.CanExecuteQuery, which is a method.

Called when "can execute" changes for any parameter

Action<CommandCanExecuteArgs> Toubab.Beinder.Tools.ICommandSource.CanExecuteQuery

Event used to execute a query.

This event will be bound to the Mixin.CommandMixin.CanExecuteQuery method. That method will update the CommandCanExecuteArgs, which can then read by the code that raised the event.

Whenever this event occurs, Mixin.CommandMixin.CanExecuteQuery will be called (if bound).
