Action<CommandCanExecuteArgs> Toubab.Beinder.Tools.CommandSource.CanExecuteQuery

Event used to execute a query by CanExecute

This event will be bound to the Mixin.CommandMixin.CanExecuteQuery method. That method will update the CommandCanExecuteArgs, which is then read by CanExecute to determine whether any bound System.Windows.Input.ICommand can execute.

Whenever this event occurs, Mixin.CommandMixin.CanExecuteQuery will be called (if bound).
