void Toubab.Beinder.Valve.BroadcastValve.Dispose ( )

Disposes of this instance.

Consists of the following parts:

  1. Unregistering of broadcast event handlers from IEvent instances in the valve
  2. Calling Dispose(bool) with a true parameter.

Just before these steps, the Disposing event is raised.

Note to subclass implementors: use Dispose(bool) to implement disposal.
